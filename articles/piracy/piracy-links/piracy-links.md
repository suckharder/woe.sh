# Piracy Links
> ⚠️ **Be wary when sailing the high seas.** If you're using trusted sources, along with some common sense, the risk of malware is low, but NEVER zero. Use a good [adblocker](https://github.com/gorhill/uBlock "uBlock Origin Github"). Avoid fake download buttons. Check file extensions.
> Torrenting? - Use a VPN. Actually, a VPN won't hurt if you're DDL-ing too. Don't forget to bind your torrent client to the VPN's interface (if possible).

Below is a list of some of my personal favorites. Use it responsibly. If possible, I always provide the official link. Avoid fakes.
